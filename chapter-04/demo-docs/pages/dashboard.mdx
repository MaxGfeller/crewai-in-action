# Dashboard

The dashboard is the first page you see after logging in. It gives you a quick overview of your chicken coop.

## Overview

The dashboard displays a table of your chickens with the following columns:
- Name
- Breed
- Health Status

Below the table, you can see a summary with the total number of chickens.

![Dashboard overview](/images/dashboard.png)

Tip: You can click on a chicken's name to view more details.
